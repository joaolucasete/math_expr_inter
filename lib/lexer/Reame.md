# Representation

### Example:

```json
{
  "person": {
    "name": "chiyoku",
    "age": 10,
    "loves": ["functional languages", "codando", "reasonml"]
  }
}
```

#### Tokens

```
[LEFT_BRACE; ID("person"); COLON; LEFT_BRACE; STRING("name"); COLON; STRING("chiyoku"); COMMA; STRING("age"); COLON; INT(10); COMMA; STRING("loves"); COLON; ...]
```
